# HR Attendance & Mission Control

This folder contains the execution pack for the Web-only / PWA HR attendance platform approved in the product brief.

Included artifacts:

- `schema.sql`: PostgreSQL schema with enums, tables, constraints, and reporting view.
- `api-contract.md`: Phase-oriented API contract for auth, passkeys, attendance, missions, location, notifications, and KPI.
- `sprint-plan.md`: Sprint-by-sprint delivery roadmap for the first implementation cycle.

Prototype:

- Open `hr-attendance/index.html` for the standalone RTL product blueprint and interactive attendance simulator.
